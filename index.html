Subtítulo adaptable y elegante

Este bloque utiliza CSS moderno para garantizar que el contenido se vea perfecto tanto en una pantalla 4K como en un smartphone antiguo. Gracias al uso de clamp() y flexbox, no necesitas ajustar manualmente cada tamaño de pantalla.

Es ideal para incrustaciones en Genially, Moodle o cualquier plataforma que utilice iframes, ya que se expande orgánicamente según el espacio disponible.
